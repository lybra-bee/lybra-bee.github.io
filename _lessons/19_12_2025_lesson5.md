### Пошаговый гайд по обучению LoRA

#### Шаг 1 — Подготовка датасета

- Возьми 10–30 качественных фото (себя, друга, кота, объекта)  
- Размер 512×512 или 768×768 (квадрат)  
- Разнообразие: разные ракурсы, освещение, эмоции, фоны  
- Удали плохие фото (размытые, слишком тёмные)  

**Инструмент для массового ресайза:** https://birme.net

#### Шаг 2 — Каптионы (текстовые описания)

Для каждого фото создай .txt файл с тем же именем:

<div class="code-block">
<pre><code class="language-text">lybragirl, девушка с каштановыми волосами, голубые глаза, улыбка, портрет
</code></pre>
</div>

**Триггер-слово:** lybragirl (уникальное, чтобы модель не путала)

#### Шаг 3 — Запуск обучения в Kohya SS

В ноутбуке выше:

<div class="code-block">
<pre><code class="language-text">- Model: stabilityai/stable-diffusion-xl-base-1.0
- LoRA type: Standard
- Network Rank: 32 (для лица) или 64 (для стиля)
- Network Alpha: 16–32
- Epochs: 10–20
- Batch size: 2–4
- Learning rate: 1e-4
- Resolution: 768
- Caption extension: .txt
- Trigger word: lybragirl
</code></pre>
</div>

Обучение на T4 займёт 15–40 минут.

#### Шаг 4 — Генерация в своём стиле

Скачай .safetensors файл и используй в любом интерфейсе:

<div class="code-block">
<pre><code class="language-text">prompt: lybragirl in cyberpunk city, neon lights, rain, cinematic, 8k
<lora:lybragirl:1.0>
</code></pre>
</div>

<br>

### 30+ советов для идеального LoRA

- Для лица: 15–25 фото, триггер + "portrait"  
- Для стиля: 20–50 картинок в одном стиле  
- Для объекта: 10–20 фото с разных ракурсов  
- Регуляризация: обязательно включи  
- LR Scheduler: cosine with restarts  
- Mixed precision: fp16  
- Save every N epochs: 5
